# EcoTrack

API REST pour la gestion écologique : suivi des activités, calcul de l'impact environnemental, gestion des utilisateurs.

## Fonctionnalités principales
- Gestion des utilisateurs
- Suivi des activités écologiques
- Calcul et affichage de l'impact environnemental
- Authentification et sécurité

## Démarrage rapide
Voir `requirements.txt` pour les dépendances et instructions d'installation.
